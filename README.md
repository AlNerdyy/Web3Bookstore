# 📚 Web3 Bookstore (Ledger & Lore)

A decentralized bookstore where authors can upload, mint, sell, and users can purchase book NFTs using Ethereum (Sepolia testnet).

---

## 🚀 Features

### 1. Upload Module
- Upload book (PDF) + cover image
- Store files on IPFS via Pinata
- Mint NFT with metadata

### 2. Selling Module
- List minted books for sale
- Set price in ETH

### 3. Purchase Module
- View all listed books
- Buy NFTs using Sepolia ETH

### 4. My Library
- View NFTs owned by your wallet
- Access book files and metadata

---

## 🛠 Tech Stack

- **Frontend:** React + Vite
- **Routing:** React Router
- **Blockchain:** Solidity (ERC-721)
- **Web3:** Ethers.js
- **Storage:** IPFS (Pinata)
- **Wallet:** MetaMask
- **Deployment:** Vercel
